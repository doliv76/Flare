How to Run Flare

Installation is different for each platform. Check out this guide for the latest React Native install instructions.

  1. Install NodeJS from this website
  2. Install React Native. Run npm install -g react-native-cli
  3. Install the dependencies. Run npm install from the project folder
  4. Link the native code. Run react-native link
  5. If you're on mac, install Xcode and the Xcode command line tools. Then run react-native run-ios
  6. If you're on windows or linux, install Android Studio, Android SDK 28, Google APIs Intel x86 Atom System Image, set $ANDROID_HOME to your sdk location, add c:\Users\YOUR_USERNAME\AppData\Local\Android\Sdk\platform-tools to your system path, open up an Android Virtual Device, and run react-native run-android

If you're running this several months after submission (in May of 2019), the Firebase server has likely been taken down. You'll need to create a new Firebase project using firebase init with support for Cloud Firestore, Authentication, and Cloud Functions. Then run firebase deploy to deploy to your new project. Finally, change index.js to use your new Firebase API key and project information.
